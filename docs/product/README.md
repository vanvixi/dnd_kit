# Product Docs

These files are the living product contract for `dnd_kit`.

- `overview.md`
- `package-architecture.md`
- `api-principles.md`
- `release-roadmap.md`

`SPEC.md` and `SPEC_JASPR.md` are retained as seed inputs and historical
references. New work should update these smaller product docs, story packets,
validation proof, and decision records instead of extending the monolithic
specs.
